Simple webserver implemented using python

How To run:
python server.py
python server.py -p 'port number'
pythonX server.py 
pythonX server.py -p 'port number'

After starting the server Go to a browser and type http://localhost:9001


Known Bugs:
	Depending on the connection, the GET request displays html page with tags. This can be fixed by refreshing the page
